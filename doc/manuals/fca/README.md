# 'ForCus AI' 사용자 매뉴얼

'ForCus AI'는 음성봇과 언어 이해(NLU) 모델을 만들고 학습·배포하며, 봇 대화 이력과 실시간 현황을 추적하고 봇·캠페인 통계를 확인하는 영역입니다. 각 화면의 캡처와 함께 사용 방법을 안내합니다.

> 작성 기준: 2026-06-02 · 기준 커밋 0e361bd9

## 봇 목록

![봇 목록](images/bot-config-bot-list__base.png)

음성봇(서비스)을 만들고 관리하는 화면입니다. 등록된 봇이 카드 형태로 표시되며, 봇마다 버전과 연결된 언어 이해(NLU) 모델, 대화 수, 서비스 개시일을 한눈에 확인할 수 있습니다. 봇 이름으로 검색하고, **추가** 버튼으로 새 봇을 만들며, 카드를 선택해 상세·수정 화면으로 이동하거나 더보기 메뉴로 복제·삭제합니다.

---

## 봇 추가

![봇 추가](images/bot-config-bot-create__base.png)

새 봇을 단계별로 만드는 화면입니다. '기본 정보'에서 봇 이름과 설명, 사용할 언어 이해(NLU) 모델, 의도 인식 신뢰도 범위, 태그를 입력하고, '음성 인식·합성 설정'에서 음성을 텍스트로 변환(STT)하는 방식과 텍스트를 음성으로 변환(TTS)하는 방식·발화자·속도·볼륨·피치를 설정한 뒤 저장합니다. 오른쪽 '입력 정보 요약'에서 진행 상황을 확인할 수 있습니다.

| 항목 | 설명 |
| --- | --- |
| 봇 이름 | 봇 이름 (필수) |
| NLU 모델 | 봇이 사용할 언어 이해(NLU) 모델 (필수) |
| 신뢰도 | 봇이 사용자 의도를 인정하는 신뢰도 하한·상한 범위 |
| 태그 | 봇 분류용 태그 |
| 음성 인식(STT) / 음성 합성(TTS) | 음성↔텍스트 변환 방식과 발화자·속도·볼륨·피치 |

| 탭 | 내용 |

| 기본정보 | 모델 이름·설명, 의도·개체 내보내기, 삭제 |

| 스냅샷 | 특정 시점의 모델 상태 저장·복원 |

## 의도 상세

![의도 상세](images/bot-config-model-_modelId-intent-_intentId__detail.png)

하나의 의도(사용자가 말하려는 목적)를 자세히 편집하는 화면입니다. 모델 상세의 '의도' 탭에서 의도를 더블클릭하면 열립니다. '기본정보' 탭에서 의도 이름과 설명을 수정하고, '의도문장' 탭에서 이 의도로 인식할 예문(학습 문장)을 추가·수정·삭제합니다.

## 개체 상세

![개체 상세](images/bot-config-model-_modelId-entity-_entityId__detail.png)

하나의 개체(전화번호·날짜·상품명처럼 사용자 발화에서 뽑아낼 정보)를 자세히 편집하는 화면입니다. 모델 상세의 '개체' 탭에서 개체를 더블클릭하면 열립니다. '기본정보' 탭에서 개체 이름과 설명을 수정하고, '유사어' 탭에서 같은 개체로 인식할 표현(유사어·동의어)을 관리합니다.

---

## 평가 상세

![평가 상세](images/bot-config-model-_modelId-evaluation-_evalId__detail.png)

모델의 인식 성능을 측정하는 평가셋 하나를 자세히 보는 화면입니다. 모델 상세의 '평가' 탭에서 평가셋을 더블클릭하면 열립니다. '기본정보' 탭에서 평가셋 이름을 수정하고, '평가 문장' 탭에서 평가에 사용할 질문과 정답을 관리하며, '평가 결과' 탭에서 평가 실행 결과(인식 정확도 등)를 확인합니다.

## FAQ 상세

| 항목 | 설명 |
| --- | --- |

| 상태 | 활성 여부 |
| 수정일 | 마지막 수정 일시 |

## 봇 대화 이력

![봇 대화 이력](images/tracking-bot-dialog-history__base.png)

봇이 고객과 나눈 통화·대화 기록을 조회하는 화면입니다. 기간과 봇·의도·신뢰구간·완료여부·발신번호·통화식별자(UCID) 등 다양한 조건으로 검색하고, **대화여정** 버튼으로 선택한 통화의 대화 흐름을 자세히 보거나 **Export** 버튼으로 결과를 내려받습니다. 각 행의 **UCID 복사** 버튼으로 통화식별자를 복사할 수 있습니다.

| 항목 | 설명 |
| --- | --- |
| 검색일자 | 조회할 시작·종료 날짜·시각 |
| 봇 / 의도 / 신뢰구간 | 봇·의도·인식 신뢰도 범위로 필터 |
| 완료여부 / 상담사연결 | 대화 완료 여부와 상담사 연결 여부 |
| 발신번호 / UCID | 발신 번호·통화식별자(UCID)로 검색 |
| 통화시간 / 신뢰도 / 재학습 | 통화 길이, 의도 인식 신뢰도, 재학습 반영 여부 |

---

## 실시간 봇 트래킹

![실시간 봇 트래킹](images/tracking-bot-realtime__base.png)

지금 진행 중인 봇 통화를 실시간으로 모니터링하는 화면입니다. 발신번호로 검색하고 **시작** 버튼으로 모니터링을 시작하면, 진행 중인 통화의 봇·통화식별자(UCID)·발신/착신번호·통화시작·통화시간이 표시됩니다.

---

## 개인정보 열람 이력

![개인정보 열람 이력](images/tracking-decrypt-log__base.png)

마스킹된 개인정보를 풀어 본(열람한) 기록을 조회하는 화면입니다. 누가 언제 어떤 정보를 어떤 사유로 열람했는지 확인할 수 있으며, 열람일자·봇·결과·사유·열람자·통화식별자(UCID)·IP 등으로 검색합니다.

| 항목 | 설명 |
| --- | --- |
| 열람 시각 / 결과 | 열람 일시와 성공/실패 |
| 열람자 | 열람한 사용자 계정·이름 |
| 봇 서비스 / UCID | 열람 대상 봇과 통화식별자(UCID) |
| 발신 / 착신 / 버블 / 화자 | 열람한 통화 정보 항목 |
| 사유 / IP | 열람 사유와 접속 IP |

## 봇 현황

![봇 현황 대시보드](images/dashboard-call-bot__base.png)

봇 운영 현황을 여러 위젯으로 한눈에 보여주는 대시보드입니다. 봇 해결률, 대화 완료율, 슬롯 완료율과 봇별 순위 등을 확인할 수 있습니다. 상단 **봇 선택**으로 특정 봇 기준의 현황을 보고, **화면편집** 버튼으로 위젯 구성을 바꾸며, 각 위젯의 버튼으로 차트와 표 보기를 전환합니다.

| 위젯 | 내용 |
| --- | --- |
| 봇/대화/슬롯 점유 현황 | 봇별 점유 순위 |
| 봇 현황 | 봇 해결률(해결·미해결 종료·상담사 연결) 요약 |
| 대화 현황 / 슬롯 현황 | 대화·슬롯의 완료/미완료 비율 |
| 미완료율 순위 / 슬롯 재질문 순위 | 봇별 미완료율, 슬롯 평균 제시도·분포 순위 |

---

## 캠페인 현황

![캠페인 현황 대시보드](images/dashboard-call-campaign__base.png)

아웃바운드(발신) 캠페인의 진행 현황을 보여주는 대시보드입니다. 캠페인 진행률, 본인 통화 완료율, 실시간 발신 건수 등을 확인할 수 있습니다. 상단 **캠페인 선택**·**시나리오 선택**으로 대상을 고르고, **화면편집** 버튼으로 위젯 구성을 바꿉니다.

| 위젯 | 내용 |
| --- | --- |
| 진행률 | 발신 시도 대비 진행 비율 |
| 본인 통화 완료율 / 검증 실패율 | 본인 통화 완료·본인 확인 실패 비율 |
| 발신 진행 건수 / 분당 발신 시도 건수 | 실시간 진행 건수와 발신 속도 |
| 현황 | 본인확인·통화완료·재시도·실패·부재·문자 발송 건수 |

---

> 아래 통계 화면들은 모두 같은 방식으로 동작합니다. 상단에서 조회 기간과 대상(봇·모델 등), 제외요일·공휴일 제외 같은 조건을 설정하고 **조회**를 누르면 결과 표가 채워지며, **Export** 버튼으로 결과를 엑셀로 내려받습니다. 표시되는 항목(열)만 화면마다 다릅니다.

## 봇(서비스) 통계

![봇(서비스) 통계](images/statistics-call-bot-service__base.png)

봇(서비스) 단위로 진입·완료·상담사 연결 현황을 날짜별로 집계해 보여주는 통계 화면입니다.

| 항목 | 설명 |
| --- | --- |
| 진입수 / 완료건 / 미완료건 | 봇 진입 수와 대화 완료·미완료 건수 |
| 완료율 / 미완료율 | 완료·미완료 비율 |
| 진입별·완료별·미완료별 상담연결 | 상황별 상담사 연결 건수와 비율 |
| 질의수 | 봇에 들어온 질의 수 |

## 대화 통계

![대화 통계](images/statistics-call-bot-dialog__base.png)

봇의 개별 대화(시나리오) 단위로 진입·완료 현황을 집계해 보여주는 통계 화면입니다. 봇뿐 아니라 대화명으로도 필터할 수 있으며, 날짜·봇·대화명별 진입수와 완료/미완료 건수·비율을 확인합니다.

## 슬롯 통계

![슬롯 통계](images/statistics-call-bot-slot__base.png)

봇 대화에서 정보를 받아내는 단위인 **슬롯**별로 수집 성공·실패와 재질문(재시도) 현황을 집계해 보여주는 통계 화면입니다. 봇·대화명·슬롯명·슬롯 유형으로 필터합니다.

| 항목 | 설명 |
| --- | --- |
| 슬롯명 / 슬롯타입 / 개체 태그 | 슬롯 이름과 종류, 연결된 개체 |
| 진입수 / 완료건 / 미완료건 | 슬롯 진입·수집 성공·실패 건수 |
| 완료율 / 미완료율 | 수집 성공·실패 비율 |
| 재시도 횟수 (1회·2회·3회 이상) | 같은 슬롯을 다시 물어본 횟수 분포 |

봇 대화에서 운영자가 직접 정의한 항목(카테고리)별로 집계해 보여주는 통계 화면입니다. 봇·카테고리·대화명으로 필터해 원하는 기준의 집계 결과를 확인합니다.

## 의도 통계

![의도 통계](images/statistics-nlu-intent__base.png)

언어 이해(NLU) 모델이 사용자 발화에서 **의도**를 얼마나 자주·정확하게 인식했는지 집계해 보여주는 통계 화면입니다. 모델·의도명으로 필터합니다.

| 항목 | 설명 |
| --- | --- |
| 검출횟수 | 해당 의도가 인식된 횟수 |
| 평균 신뢰도 | 인식 신뢰도 평균 |
| 신뢰도 성공 / 재확인 / 실패 | 신뢰도 구간별 인식 건수(성공·재확인 필요·실패) |

## 개체 통계

![개체 통계](images/statistics-nlu-entity__base.png)

언어 이해(NLU) 모델이 발화에서 **개체**(전화번호·날짜·상품명 등)를 어떤 값으로 얼마나 자주 추출했는지 집계해 보여주는 통계 화면입니다. 모델·개체 태그로 필터하며, 개체 태그·개체 값별 검출횟수를 확인합니다.

## 키워드 통계

![키워드 통계](images/statistics-nlu-keyword__base.png)

사용자 발화에서 자주 나온 **키워드**를 집계해 보여주는 통계 화면입니다. 모델로 필터하며, '개체 태그 미분류' 옵션으로 개체로 분류되지 않은 키워드만 따로 볼 수 있습니다.
